微软将要添加Linux命令行到Windows 10系统

译文

Windows将要迎来Bash Shell，对开发者，开发者，开发者来说是个大新闻。
原文链接 Microsoft is adding the Linux command line to Windows 10
原文作者 Chris Welch
译者 Ancoka
校对者 Ancoka

微软正在以一种该公司前所未有的方式拉拢Linux开发者。Kevin Gallo(凯文·加洛)在微软的2016 build 开发者大会的舞台上说：“Windows系统将要迎来Bash Shell。是的，Windows系统将要迎来真正的Bash Shell”。这个公告收到了人群中热烈的掌声。新功能将要做为今年夏天周年纪念的一部分开启更新到Windows 10系统。
“这不是一个虚拟机。这不是交叉编译工具，这是原生的，”他说。“我们已经与提供这个伟大的经历的Canonical公司展开合作，你就可以直接从Windows应用商店下载。”第三方已经占有这项功能多年了，但是微软与Canonical公司的直接合作关系应向那些比较喜欢这些二进制文件和工具的开发者提供更多的灵活性和便利性。
更重要的是，它代表了微软对开源开发的耳目一新的立场。副总Terry Myerson（裁特里·迈尔森）戏弄称：关于其他更多可能性的“东西将要到来“，标志着Windows 10更加现在化和开放。本博客文章取自微软Scott（斯科特）Hanselman，详细内容请移步到Scott Hanselman’s Developers can run Bash Shell and user-mode Ubuntu Linux binaries on Windows 10。“对于像我这样使用多种工具集的开发者来说，这是的超赞的，”他说。 “这是我在Windows之上使用的所有的Linux工具中，真正像在Ubuntu系统上的一款工具。”

更新：微软已发布了包含Bash在Windows上一段讲解非常深入的视频，我们已经嵌入在本文的顶部。它应该回答你所有的疑问。
